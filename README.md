Meetify is a platform for managing onboarding, training, consultations, and events where all interactions are handled through the system instead of direct communication between participants.

## Features
- Host registration
- Slot creation and management
- Slot discovery and booking
- Stripe payment integration
- Booking confirmation flow
- Full-stack integration (Frontend + Backend)

## Tech Stack
- Frontend: Next.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Payments: Stripe API
